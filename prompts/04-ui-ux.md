# Prompt: UI and UX

Help design or implement the UI for this engineering knowledge base.

## Desired style

- simple
- clean
- technical
- quiet visual design
- easy to scan
- no marketing fluff
- no fancy animations unless there is a real usability benefit

## Home page

The home page should focus on:
- a consistent header
- a search bar
- visible tags
- fast access to content

## Navigation philosophy

- Keep navigation shallow.
- Use tags and search rather than deep menu trees.
- Make formulas and calculators easy to discover.

## Page design guidance

### Formula pages
Clearly separate:
- formula display
- variables
- units
- assumptions
- explanation
- examples

### Calculator pages
Clearly separate:
- purpose
- inputs
- outputs
- units
- worked examples
- related formulas/notes

## Implementation preference

Favor readability and maintainability over visual novelty.
