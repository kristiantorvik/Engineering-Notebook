# Prompt: Project Overview

You are helping me build an engineering knowledge base website.

## Goal

Create a simple, static, searchable website that lets me store and browse:
- notes
- formulas
- calculators
- reference tables/charts

This site is meant to replace scattered paper notes, bookmarks, online calculators, and formula sheets.

## Audience

Primary audience:
- me
- colleagues
- friends

It is not intended to be a large public product or a marketing site.

## Main requirements

- Static site hosted with GitHub + Cloudflare Pages
- Simple to edit locally in VS Code
- Simple to test locally
- Simple to deploy by pushing changes
- Cheap to operate
- Tag-based discovery
- Minimal navigation complexity
- Clean, technical UI

## Technical direction

- MkDocs + Material for MkDocs
- Markdown with frontmatter for content
- Math rendering for formulas
- Client-side calculators
- Python tooling/scripts allowed and encouraged

## Important design principle

Prefer simple, maintainable, content-first implementations.
Do not overengineer version 1.
