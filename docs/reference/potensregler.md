# Potensregler:

$\Large a^x \cdot a^y = a^{x+y}$

$\,$

$\Large \dfrac{a^x}{a^y} = a^{x-y}$

$\,$

$\Large a^{-x} = \dfrac{1}{a^x}$

$\,$

$\Large (a^x)^y = a^{x \cdot y}$

$\,$

$\Large a^{\frac{x}{y}} = \sqrt[y]{a^x}$
