# Logarimteregler:

$\Large ln(a \cdot b) = ln(a) + ln(b)$

$\,$

$\Large ln(\dfrac{a}{b}) = ln(a) - ln(b)$

$\,$

$\Large ln(a^b) = b \cdot ln(a)$

$\,$

$\Large log_a(b) = \dfrac{ln(b)}{ln(a)}$
